Driver Napalm

Language : Python
Librairies : Napalm, Request, Netmiko, Pexpect, Pytest, Pylint, Black, Bandit

Lors de mon stage et de mon alternance, j'ai eu l'occasion de développer plus d'une dizaine de driver Napalm pour l'automatisation de la configuration des équipements réseau de BPCE Infogérance & Technologies.

Un driver Napalm est un composant de NAPALM (Network Automation and Programmability Abstraction Layer with Multivendor support), une bibliothèque Python utilisée pour automatiser et interagir avec différents appareils et systèmes d'exploitation réseau. NAPALM offre une API unifiée pour gérer des appareils de différents fabricants, facilitant ainsi l'automatisation réseau indépendamment du fournisseur. Chaque driver dans NAPALM est conçu pour interagir avec un appareil réseau ou un système d'exploitation spécifique, permettant l'exécution de commandes et opérations cohérentes sur diverses plateformes.

Plugin Nautobot

Language : Python
Librairies : Napalm, Nornir, Django

Lors de mon stage et de mon alternance, j'ai eu l'occasion de développer un plugin Nautobot pour l'automatisation de l'inventaire des équipements réseau de BPCE Infogérance & Technologies.

Nautobot est un outil open-source pour la gestion des réseaux et data centers, offrant une interface utilisateur et une API RESTful pour l'automatisation et l'intégration. Les plugins Nautobot sont des extensions qui ajoutent des fonctionnalités spécifiques, comme l'intégration avec d'autres systèmes, sans modifier le code source de Nautobot, permettant une personnalisation et une extension adaptées aux besoins des utilisateurs.

Plugin Centreon

Language : Perl
Communications : API, SSH, SNMP

Lors de mon stage et de mon alternance, j'ai eu l'occasion de développer plusieurs plugins Centreon pour la supervision des équipements réseau de BPCE Infogérance & Technologies.

Centreon est une plateforme de surveillance réseau qui permet aux entreprises de surveiller en temps réel leur infrastructure informatique. Elle offre une vue centralisée des systèmes et réseaux, facilitant la détection et la prévention des problèmes. Extensible via des plugins, Centreon est personnalisable pour répondre aux besoins spécifiques de chaque environnement informatique.
